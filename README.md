# Server Store

A simple, configurable static store site.

## Configuration

Edit `config.yml` to change:
- Shop Titles and Descriptions
- WhatsApp Number and Seller Name
- Products (Images, Prices, Descriptions)

## Deployment on Vercel

1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the project into Vercel.
3. Deploy! (No build command needed, it is a static site).

## Images
Place your images in the `assets` folder and reference them in `config.yml`.
